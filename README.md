# Bank-System

This project is a Bank Management System developed in C++, designed to simulate the operations of a real-world bank. It provides a console-based interface for managing multiple customer accounts and handling banking transactions efficiently.

Features

Customer Account Management: Create, update, and delete customer accounts with personal and account details.

Deposit & Withdrawal: Perform secure deposit and withdrawal operations for each account.

Balance Inquiry: Check the balance of any customer account.

Transaction History: Maintain a record of all transactions for auditing purposes.

File Storage: Account data and transactions are stored in files to preserve data across sessions.

User-Friendly Menu: Easy navigation through a command-line interface for performing banking operations.

Technologies

C++ programming language

File handling for persistent data storage

Object-Oriented Programming (OOP) concepts: classes, inheritance, encapsulation, and modular design
